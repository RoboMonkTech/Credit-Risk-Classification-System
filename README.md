# 💳 Credit Risk Classification System Using Machine Learning

An AI-powered web application that predicts whether a loan applicant has **Low Credit Risk** or **High Credit Risk** using a Random Forest Machine Learning model.

## 📌 Project Overview

This project analyzes customer financial and personal information to classify credit risk. It assists financial institutions in making informed lending decisions.

## 🚀 Features

- Credit Risk Prediction
- Random Forest Classifier
- User-Friendly Web Interface
- Business Decision Support
- Interactive Dashboard

## 📊 Dataset

German Credit Risk Dataset

### Input Features

- Age
- Sex
- Job
- Housing
- Saving Accounts
- Checking Account
- Credit Amount
- Duration
- Purpose

### Output

- 🟢 Low Credit Risk
- 🔴 High Credit Risk
